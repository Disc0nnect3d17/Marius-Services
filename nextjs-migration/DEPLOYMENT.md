# Pro Fit Bathroom and Kitchen - Next.js

Modern, fast, and SEO-friendly website built with Next.js.

## Quick Start

```bash
npm install
npm run dev
```

## Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new)

Click the button above or:

```bash
npm install -g vercel
vercel
```

Your site will be live in seconds!
